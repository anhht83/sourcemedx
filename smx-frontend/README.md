# smx-frontend

Frontend application of SourceMedX

## Global Requisites

* nextjs (^15.*)

## Install, Configure & Run

Below mentioned are the steps to install, configure & run in your platform/distributions.

### Install

```bash
# Run command to clone the repo.
git clone https://github.com/SourceMedX/smx-frontend.git

# Goto the cloned project folder, run command
cd smx-frontend

# Run command to install packages dependencies.
npm install
```
